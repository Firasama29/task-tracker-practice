This project is intended as a practice to build a task tracker Command Line Interface (CLI), as part of projects recommended by [roadmap.sh](https://roadmap.sh/projects/task-tracker).

This task tracker project should be good for practising programming skills and tasks like working with filesystem, handling user inputs and building a simple CLI app.

## Requirements
As instructed in roadmap.sh, the app should run from the command line, accept user actions and inputs and store the tasks in a JSON file. The following are the activities a user can perform:

- Add, update, delete a task
- Mark a task as in-progress or done
- List all tasks
- List all completed tasks
- List tasks that are not done (open, pending, in-progress)

## Constraints
- Store tasks in JSON file
- If JSON file does not exit, create it
- No frameworks on external libraries should be used
- Errors should be handled gracefully


The project will be developed in two different versions, one in Java and the other with python.
